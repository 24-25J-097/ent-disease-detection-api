Authentication

#

Login

#
POST /api/public/login

Authenticate a user and receive a JWT token for API access.

Request Body

{
  "email": "doctor@example.com",
  "password": "password123"
}

Response

{
  "user": {
    "id": "user_123",
    "email": "doctor@example.com",
    "role": "doctor"
  },
  "token": "eyJhbGciOiJIUzI1NiIs..."
}

Patients

#

Create Patient

#
POST /api/doctor/patients

Create a new patient profile with comprehensive medical information.

Request Body

{
  "firstName": "John",
  "lastName": "Doe",
  "email": "john.doe@email.com",
  "phone": "+1234567890",
  "dateOfBirth": "1990-05-15",
  "gender": "male",
  "address": "123 Main St, City"
}

Response

{
  "patient": {
    "id": "patient_789",
    "firstName": "John",
    "lastName": "Doe",
    "email": "john.doe@email.com",
    "createdAt": "2024-01-15T10:30:00Z"
  }
}

Diagnosis

#

Cholesteatoma Diagnosis

#
POST /api/doctor/diagnosis/cholesteatoma

Submit an endoscopy image for AI-powered cholesteatoma diagnosis analysis.

Content-Type: multipart/form-data

Form Data

diagnosticianId (string, required)
patientId (string, required)
endoscopyImage (file, required)
additionalInformation (string, optional)

Response

{
  "diagnosis": {
    "id": "diag_123",
    "patientId": "patient_789",
    "condition": "cholesteatoma",
    "confidence": 0.92,
    "result": "positive",
    "imageUrl": "/api/diagnosis/image/diag_123",
    "createdAt": "2024-01-15T15:30:00Z",
    "status": "pending_review"
  }
}
